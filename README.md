# AI Recruiter – Intelligent Candidate Discovery

## Problem
Traditional recruitment relies on keyword matching, which often misses qualified candidates.

## Solution
This project uses AI to rank candidates by understanding job descriptions and candidate profiles using semantic similarity and hybrid scoring.

## Tech Stack
- Python
- Pandas
- Sentence Transformers
- Scikit-learn
